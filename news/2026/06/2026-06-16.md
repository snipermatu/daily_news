## 2. IA

- **Anthropic publicó los primeros resultados de `Anthropic Public Record`.** El **12 de junio de 2026**, Anthropic presentó una encuesta con casi **52.000 estadounidenses** para medir cómo perciben la IA. No es un lanzamiento de producto, pero sí una señal útil: el debate competitivo ya no es solo modelos y capacidad, sino también confianza pública, usos aceptables y adopción social.
- **Qué cambia en la práctica:** para equipos enterprise, esto refuerza que durante 2026 la discusión se está moviendo desde “qué modelo rinde más” hacia “qué modelo se puede desplegar con legitimidad, gobernanza y aceptación”.

## 4. Azure

- **Las `Work IQ APIs` pasan a disponibilidad general el 16 de junio de 2026.** Aunque el marco se anunció en `Microsoft Build 2026`, Microsoft especificó que esta capa programática de contexto laboral queda **GA el 16 de junio**. Es una novedad material porque habilita acceso estable a señales de trabajo reales para agentes conectados con `Microsoft Foundry`, `Fabric IQ` y conocimiento empresarial.
- **Por qué importa:** es un paso concreto desde demos hacia operación. El punto no es solo “más contexto”, sino contexto empresarial accesible por API para agentes con grounding en personas, documentos, reuniones y sistemas organizacionales.
